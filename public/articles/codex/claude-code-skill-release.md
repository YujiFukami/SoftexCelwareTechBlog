# Release Skill

正本は `CLAUDE.md` の「リリース手順」セクション。
このSkillは、Claude Codeが実行しやすい形に手順を並べたもの。

## STEP 1. 事前確認

- 作業ブランチと未コミット差分を確認する
- リリース対象のバージョンを確認する
- ユーザー操作が必要な手順を先に伝える

## STEP 2. ビルドと検証

- 規定のビルドコマンドを実行する
- テストまたは起動確認を行う
- 失敗した場合は原因と次の対応を報告する

## STEP 3. 公開準備

- 配布物を作成する
- READMEやリリースノートを更新する
- 公開前チェックリストを確認する

## STEP 4. 完了報告

- 実行したコマンド
- 生成物
- 変更ファイル
- ユーザーが次に確認すること
をまとめて報告する。
